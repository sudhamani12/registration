Selectors and Properties

Selectors

A CSS selector is the first part of a CSS Rule. It is a pattern of elements and other terms that tell the browser which HTML elements should be selected to have the CSS property values inside the rule applied to them.

                                
                            *{ 
                                color: purple; 
                              }
What are the 4 CSS selectors?

CSS Selectors

  • Simple selectors (select elements based on name, id, class)
  • Combinator selectors (select elements based on a specific relationship between them)
  • Pseudo-class selectors (select elements based on a certain state)
  • Pseudo-elements selectors (select and style a part of an element)

    
*h1 { 
    text-transform: uppercase; 
  }
What is the difference of selector property and value?

A property is an aspect of a selector. For instance, you can change the font-family, color, and font-size of the text on your web pages (in addition to many more). A value is a possible setting for a property.

Which selector is best in CSS?

class selector. The CSS class selector is probably the most commonly used selector. It applies styling to all elements with a specified class attribute. It's a great way to customize how CSS rules apply by creating attributes that can be applied to any element to give it a certain styling.

What are type selectors?

A Type Selector (sometimes referred to as an Element Type Selector) matches elements with the corresponding element node name, such as <p>, <span> , and <div> tags. Type selectors are generally used to make “broad stroke” changes to the style of a site.

How many properties can a selector have?

There are many properties and their values for HTML selectors. According to CSS Tricks, there are "520 distinct property names from 66 technical reports and 66 editor's drafts". These properties are common because they are frequently used in all CSS documents and can be applied to different selectors.

A CSS property assign a style or behavior to an HTML element. Examples include: color, border, margin, font-style, and transform.

The all shorthand CSS property resets all of an element's properties except unicode-bidi , direction , and CSS Custom Properties. It can set properties to their initial or inherited values, or to the values specified in another cascade layer or stylesheet origin.

The content CSS property replaces an element with a generated value. Objects inserted using the content property are anonymous replaced elements.

CSS value definition syntax, a formal grammar, is used for defining the set of valid values for a CSS property or function. In addition to this syntax, the set of valid values can be further restricted by semantic constraints (for example, for a number to be strictly positive).

properties and rules of CSS

A CSS rule is a grouping of one or more CSS properties which are to be applied to one or more target HTML elements. A CSS rule consists of a CSS selector and a set of CSS properties. The CSS selector determines what HTML elements to target with the CSS rule.
